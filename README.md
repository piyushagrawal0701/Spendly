# 💸 Spendly – Personal Finance Tracker

Spendly is a modern personal finance tracking web app built with React.  
It helps users manage income, expenses, and balance with a clean UI, charts, and interactive elements.

---

## 🚀 Features

### 📊 Dashboard
- Total Balance, Income, Expenses overview
- Clean card-based UI

### 🧾 Transactions
- Add Income & Expense
- Category & Date support
- Emoji-based icon selection 🎨

### 📈 Analytics
- Pie chart visualization using Recharts
- Custom tooltip & legend

### 🧠 State Management
- Context API for global transaction handling
- Clean and scalable structure

### 💾 Local Storage
- Data stored in browser (no backend)
- Auto sync on updates

### 🎮 Interactive UI
- Matter.js powered "Magic Box" visualization

### 🧹 Reset Feature
- Clear all transactions with confirmation + toast

### 📥 Export Data
- Download all transactions as CSV

### 📱 Responsive Design
- Works on mobile & desktop

---

## 🛠️ Tech Stack

- React.js

- React Icons
