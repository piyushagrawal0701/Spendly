💸 Spendly – Personal Finance Tracker

Spendly is a modern and interactive personal finance tracking web app built using React. It helps users track income, expenses, and overall balance with beautiful UI components and real-time visualizations.

🚀 Features

📊 Dashboard Overview
Total Balance, Income, and Expenses
Clean and responsive UI cards
🧾 Transaction Management
Add Income & Expense entries
Category & date support
Emoji-based icon selection 🎨
🔄 Local Storage Persistence
Data is saved in browser (no backend needed)
📈 Charts & Analytics
Pie charts for financial breakdown
Custom legends & tooltips
🧠 Context API State Management
Global transaction handling
Clean and scalable architecture
🎮 Interactive Physics UI
Matter.js powered "Magic Box" visualization
🧹 Reset All Transactions
Clear all data with confirmation & toast feedback
📱 Fully Responsive
Works perfectly on mobile & desktop

🛠️ Tech Stack

⚛️ React.js
🎨 Tailwind CSS
📊 Recharts
🎯 React Icons
😊 Emoji Picker
⚡ Matter.js (for physics animation)
🍞 React Hot Toast

⚙️ Installation & Setup
# Clone the repo
git clone https://github.com/piyushagrawal/spendly.git

# Navigate to project
cd spendly

# Install dependencies
npm install

# Run project
npm run dev
